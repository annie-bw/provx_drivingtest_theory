# Driving Theory Test Preparation

React + TypeScript + Vite frontend for a Kinyarwanda-based driving theory practice and exam platform.

## Features
- Landing page in Kinyarwanda
- Student login flow
- Practice mode with 20 questions and instant feedback
- Exam mode with 20-minute timer and tracked results
- Result and review screens
- Admin dashboard UI for managing question banks
- Tailwind CSS styling with a modern SaaS tone

## Setup
1. `cd C:\Users\delta\driving-theory-prep`
2. `npm install`
3. `npm run dev`

## Notes
- This is a frontend scaffold only. Backend and database integration can be added next.
